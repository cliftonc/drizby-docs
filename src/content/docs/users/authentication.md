title: Authentication

---

Drizby supports 7 authentication methods:

- **Google OAuth**
- **GitHub OAuth**
- **GitLab OAuth**
- **Microsoft OAuth**
- **Slack OAuth**
- **Magic Link** (email-based)
- **Password** (local accounts)

## First-Time Setup

On first launch, Drizby prompts you to create an admin account with email and password. You can then configure OAuth providers from the Settings page.
