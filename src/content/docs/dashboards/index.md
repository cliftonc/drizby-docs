Drizby includes a full dashboard builder with drag-and-drop grid layout, resizable widgets, and a variety of chart types.

## Chart Types

- Bar, Line, Area, Pie, Scatter
- KPI / Metric cards
- Data tables
- Heatmaps

## Building a Dashboard

1. Create a new dashboard from the Dashboards page.
2. Add portlets using the analysis builder or by configuring chart queries directly.
3. Drag and resize portlets on the grid to arrange your layout.
4. Dashboards auto-save and can be shared with your team.
